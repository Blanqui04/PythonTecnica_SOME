# PythonTecnica SOME - Guia d'Instal·lació

## Requisits previs

1. **Python 3.8 o superior** instal·lat al sistema
   - Descarregar de: https://www.python.org/downloads/
   - ⚠️ **IMPORTANT**: Marcar "Add Python to PATH" durant la instal·lació

2. **Git** (opcional, només si clones el repositori)

## Instal·lació Ràpida

### Opció 1: Clonar repositori (recomanat)
```bash
git clone https://github.com/Blanqui04/PythonTecnica_SOME.git
cd PythonTecnica_SOME
```

### Opció 2: Descarregar ZIP
1. Descarregar el repositori com a ZIP
2. Extreure'l a una carpeta
3. Obrir terminal a la carpeta

## Configuració automàtica

1. Executar `SETUP.bat` (doble clic o des de terminal)
2. Esperar que s'instal·lin totes les dependències
3. Quan acabi, ja està llest per usar!

## Executar l'aplicació

- Executar `RUN_APP.bat` (doble clic)
- L'aplicació s'obrirà automàticament

## Resolució de problemes

### Error: "Python no està instal·lat"
- Instal·lar Python des de python.org
- Assegurar-se que està al PATH del sistema

### Error: "Entorn virtual no trobat"
- Executar primer `SETUP.bat`

### Error durant la instal·lació de paquets
- Comprovar connexió a internet
- Executar com a administrador si cal

## Estructura del projecte

```
PythonTecnica_SOME/
├── SETUP.bat           # Configuració inicial
├── RUN_APP.bat         # Executar aplicació
├── requirements.txt    # Dependències
├── main_app.py        # Aplicació principal
└── src/               # Codi font
```

## Suport

Per problemes o dubtes, contactar amb l'administrador del sistema.
